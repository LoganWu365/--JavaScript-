1. **概念**：栈是一个后进先出的数据结构（因为js中没有栈，所以用数组来模拟）】
2. **用法**：stack.push()入栈
            stack.pop()出栈
            stack[stack.length - 1] 栈顶元素
3. **适用场景**：需要用到**后进先出**的场景
例如： 
+ 十进制转二进制：后进的的余数反而排在前面
+ 括号合法性：同一种括号，越靠后的左括号，对应的右括号靠前。通过遍历括号数组，遇到左括号入栈，遇到右括号出栈，最后栈空了即为合法
+ **函数调用堆栈**（后面有空处理）
